# miniLab1 — Sistema de NPS com Análise por IA

## Descrição

Sistema completo de coleta e análise de NPS (Net Promoter Score) com classificação automática por inteligência artificial e painel de resultados em tempo real.

## Fluxo do sistema

### 1. Coleta
Formulário web simples acessível via link onde o cliente responde:
- **Nota de 0 a 10** (NPS)
- **Campo de texto livre** com o motivo da nota

### 2. Análise por IA
A IA processa automaticamente cada resposta e classifica:
- **Sentimento** — positivo, negativo ou neutro
- **Tema recorrente** — atendimento, preço, produto, entrega
- **Urgência** — prioridade de atenção da resposta

### 3. Painel de resultados
Dashboard com:
- NPS calculado em tempo real
- Temas mais citados por frequência
- Principais oportunidades de melhoria ordenadas por relevância
